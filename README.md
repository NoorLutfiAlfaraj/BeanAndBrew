# Bean & Brew ☕

Bean & Brew is an e-commerce website developed for a university web development project.

## Features
- Customer registration and login
- Product catalog
- Product details pages
- Shopping cart
- Checkout page
- Admin dashboard
- Add products
- Edit products
- Delete products
- MySQL database integration

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
