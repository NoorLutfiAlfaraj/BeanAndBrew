# ☕ Bean & Brew

## About the Project

Bean & Brew is an e-commerce website developed as part of a university web development project. The goal was to build a complete online shopping experience where customers can browse coffee products, view product details, add items to their cart, and complete a purchase. The website also includes an admin dashboard that allows administrators to manage the products available in the store.

This project helped me gain practical experience in both front-end and back-end web development by combining HTML, CSS, JavaScript, PHP, and MySQL.

---

## Features

### Customer Side

* Browse products by category
* View product details
* Add products to the shopping cart
* Update or remove items from the cart
* Checkout page
* Customer registration and login
* Contact Us page
* Responsive and user-friendly interface
* JavaScript form validation

### Admin Side

* Secure admin login
* Dashboard
* Add new products
* Edit existing products
* Delete products
* Search products
* Upload product images
* Manage product information stored in the database

## How to Run the Project

1. Install XAMPP.
2. Copy the project folder into the **htdocs** directory.
3. Start both **Apache** and **MySQL** from the XAMPP Control Panel.
4. Open phpMyAdmin and create a database named **beanandbrew**.
5. Import the provided SQL file into the database.
6. Open your browser and visit:

```
http://localhost/BeanAndBrew/
```
